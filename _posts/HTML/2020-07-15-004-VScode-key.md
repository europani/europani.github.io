Ctrl + D : 같은 텍스트 선택


Alt + 마우스클릭 : 커서 다중생성

Alt + Shift + I : 선택된 영역의 모든줄에 커서 다중생성


Alt +↑↓ : 한줄이동

Alt + Shift +↑↓ : 한줄 복사이동


Ctrl + / : 주석처리(해제시 같은키)

Ctrl + B : 사이드바 표시/제거

Ctrl + K + F : 문서전체 자동정렬

Ctrl + Alt + A : 블록주석처리 (해제시 같은키)

Ctrl + Shift + K :한줄 삭제
