### 1. Node.js 설치 (npm 포함)
```bash
$ brew install node

$ node -v
v12.21.0
$ npm -v
6.14.11
```

### 2. yarn 설치
```bash
$ npm -g install yarn

$ yarn -v
1.22.10
```

### 3. create-react-app 설치
```bash
# npm 사용시
$ npm install -g create-react-app
# yarn 사용시
$ yarn global add create-react-app
```

### 4. react 프로젝트 생성 및 실행
```bash
$ create-react-app {프로젝트명}

$ cd {프로젝트명}
$ yarn start
```
