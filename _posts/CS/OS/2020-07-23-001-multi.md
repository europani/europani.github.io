● 멀티태스킹 : 다수의 task(쓰레드, 프로그램)을 동시에 실행하는 것

 - 운영체제 스케줄링에 의해 번갈아서 처리

● 멀티프로그래밍 : 하나의 프로세서에서 다수의 프로그램을 동시에 실행하는 것

 - 프로세서의 자원낭비를 막음

● 멀티프로세싱 : 다수의 프로세서에서 다수의 프로그램을동시에 실행하는 것

● 멀티쓰레딩 : 하나의 프로세서에서 다수의 쓰레드를 돌리는 것
