[JAVA] equalsмҷҖ hashCode мһ¬м •мқҳ

#Java

мӮ¬мҡ©мһҗ м •мқҳ нҒҙлһҳмҠӨлҘј л§Ңл“Ө л•Ң hashCode(), equals(), toString() лҘј мһ¬м •мқҳн•ңлӢӨ.
м—¬кё°м„ң hashCode(), equals()лҠ” к°қмІҙмқҳ лҸҷл“ұм„ұкіј кҙҖл Ёмқҙ мһҲлӢӨ

  • лҸҷмқјм„ұ(identity) : л‘җ к°қмІҙк°Җ мҷ„м „нһҲ к°ҷлӢӨлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ
    • к°қмІҙмқҳ мқёмҠӨн„ҙмҠӨ мЈјмҶҢлҘј л№„көҗ
    • == лҘј мӮ¬мҡ©
  • лҸҷл“ұм„ұ(equality) : л‘җ к°қмІҙмқҳ м •ліҙк°Җ к°ҷлӢӨлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ (л…јлҰ¬м Ғ лҸҷл“ұм„ұ)
    • к°қмІҙ лӮҙл¶Җмқҳ к°’мқ„ л№„көҗ
    • equals()лҘј мӮ¬мҡ©
  • лӢЁ, кё°ліё нғҖмһ…(Primitive)лҠ” ==лЎң к°’мқ„ л№„көҗн•  мҲҳ мһҲлӢӨ

equals()

  • лҸҷл“ұм„ұ л№„көҗлҘј мң„н•ҙ мӮ¬мҡ©н•ҳлҠ” л©”м„ңл“ң
class Main {
    static class Person {
        String name;
        int age;

        public Person(String name, int age) {
            this.name = name;
            this.age = age;
        }
    }   
 
    public static void main(String[] args) {
        Person p1 = new Person("aoh", 20);
        Person p2 = new Person("aoh", 20);
      
        System.out.println(p1.hashCode());      // 746292446
        System.out.println(p2.hashCode());      // 1072591677
        System.out.println(p1==p2);             // false
        System.out.println(p1.equals(p2));      // false
    }
}
  • л‘җ к°қмІҙмқҳ м •ліҙк°Җ к°ҷм§Җл§Ң лӢӨлҘё л©”лӘЁлҰ¬ мЈјмҶҢлҘј к°–лҠ” к°қмІҙмқҙкё° л•Ңл¬ём—җ p1==p2м—җм„ң falseк°Җ лӮҳмҳЁлӢӨ
  • н•ҳм§Җл§Ң p1.equals(p2)м—җм„ңлҸ„ falseк°Җ лӮҳмҳӨкІҢ лҗҳлҠ”лҚ° мқҙмң лҠ” Object.equals() л•Ңл¬ёмқҙлӢӨ.
    • кё°ліём ҒмңјлЎң лҸҷмқјм„ұмқ„ л№„көҗн•ңлӢӨ
public class Object{
    ...
    public boolean equals(Object obj) {
        return (this == obj);
    }
    ...
}
  • кё°ліём ҒмңјлЎң лҸҷмқјм„ұмқ„ л№„көҗн•ҳкё° л•Ңл¬ём—җ мҳӨлІ„лқјмқҙл”©мқ„ н•ҙм„ң лҸҷл“ұм„ұ л№„көҗлЎң кё°мӨҖмқ„ мһ¬м •мқҳ н•ҙм•јн•ңлӢӨ
  • String.equals()лҠ” лҸҷмқјм„ұ л№„көҗлҘј мң„н•ҙ мһҳ м •мқҳлҗҳм–ҙ мһҲмңјлӢҲ мқҙлҘј мӮ¬мҡ©н•ҳлҠ” кІғлҸ„ мўӢлӢӨ

hashCode()

  • кё°ліём ҒмңјлЎң к°қмІҙмқҳ л©”лӘЁлҰ¬ мЈјмҶҢлҘј л°ҳнҷҳн•ҳлҠ” л©”м„ңл“ң
  • лӘЁл“  к°қмІҙлҠ” лӢӨлҘё кіөк°„м—җ н• лӢ№лҗҳкё° л•Ңл¬ём—җ лӢӨлҘё л©”лӘЁлҰ¬ мЈјмҶҢлҘј к°–кІҢ лҗңлӢӨ. мҰү, лӘЁл“  к°қмІҙлҠ” кё°ліём ҒмңјлЎң лӢӨлҘё н•ҙмӢңмҪ”л“ңлҘј к°–лҠ”лӢӨ
  • л…јлҰ¬м ҒмңјлЎң л‘җ мқёмҠӨн„ҙмҠӨк°Җ к°ҷлӢӨл©ҙ к°ҷмқҖ н•ҙмӢңмҪ”л“ңлҘј л°ҳнҷҳн•ҳлҸ„лЎқ мһ¬м •мқҳ н•ҙм•јн•ңлӢӨ

  • н•ҙмӢң мһҗлЈҢкө¬мЎ°лҠ” к°ҷмқҖ к°қмІҙмқём§Җ нҢҗлӢЁн•  л•Ң н•ҙмӢңк°’мқ„ мӮ¬мҡ©н•ҳкІҢ лҗңлӢӨ. л…јлҰ¬м ҒмңјлЎң к°ҷмқҖ мқёмҠӨн„ҙмҠӨлҘј к°ҷмқҖ к°қмІҙлЎң нҢҗлӢЁн•ҳкё° мң„н•ҙ мһ¬м •мқҳк°Җ н•„мҡ”н•ҳлӢӨ

  • мһ¬м •мқҳмӢң Objects.hash(args)лҘј мӮ¬мҡ©н•ҳл©ҙ мүҪкІҢ мһ¬м •мқҳк°Җ к°ҖлҠҘн•ҳлӢӨ
class Main {
    static class Person {
        String name;
        int age;

        public Person(String name, int age) {
            this.name = name;
            this.age = age;
        }

        @Override
        public int hashCode() {
            return Objects.hash(name, age);
        }

        @Override
        public boolean equals(Object o) {
            Person p = (Person) o;
            return (name.equals(p.name)) && age==p.age;
        }
    }   
 
    public static void main(String[] args) {
        Person p1 = new Person("aoh", 20);
        Person p2 = new Person("aoh", 20);
      
        System.out.println(p1.hashCode());    // 3000603
        System.out.println(p2.hashCode());    // 3000603
        System.out.println(p1==p2);           // false
        System.out.println(p1.equals(p2));    // true
    }
}
  • nameкіј ageмқҳ к°’мңјлЎң н•ҙмӢң мҪ”л“ңлҘј мғқм„ұн•ҳлҸ„лЎқ н•ҳкё° л•Ңл¬ём—җ к°ҷмқҖ н•ҙмӢңмҪ”л“ңлҘј л°ҳнҷҳн•ҳкІҢ лҗңлӢӨ
  • p1==p2 : л‘җ к°қмІҙлҠ” лӢӨлҘё к°қмІҙлЎңмҚЁ лӢӨлҘё л©”лӘЁлҰ¬ мЈјмҶҢк°’мқ„ к°–кё° л•Ңл¬ём—җ falseлҘј л°ҳнҷҳ
  • p1.equals(p2) : nameкіј ageмқҳ к°’мқ„ л№„көҗн•ҳлҸ„лЎқ мһ¬м •мқҳ н–Ҳкё° л•Ңл¬ём—җ trueлҘј л°ҳнҷҳ

equalsл§Ң мҳӨлІ„лқјмқҙл“ң н•ҳлҠ” кІҪмҡ°

class Main {
    static class Person {
        String name;
        int age;

        public Person(String name, int age) {
            this.name = name;
            this.age = age;
        }

        @Override
        public boolean equals(Object o) {
            Person p = (Person) o;
            return (name.equals(p.name)) && age==p.age;
        }

        @Override
        public String toString() {
            return name + ":" + age;
        }
    }   
 
    public static void main(String[] args) {
        HashSet<Person> set = new HashSet();

        set.add(new Person("aoh", 20));
        set.add(new Person("aoh", 20));
        System.out.println(set);    //  [aoh:20, aoh:20]
    }
}
  • н•ҙмӢңмҪ”л“ңк°Җ лӢӨлҘҙкё° л•Ңл¬ём—җ HashSetмқҖ лӢӨлҘё к°қмІҙлЎң мқёмӢқн•ңлӢӨ

hashCodeл§Ң мҳӨлІ„лқјмқҙл“ң н•ҳлҠ” кІҪмҡ°

  • equals()лҘј нҶөн•ҙ лҸҷл“ұм„ұмқ„ нҢҗлӢЁн•ҳлҠ”лҚ° мһ¬м •мқҳлҘј н•ҳм§Җ м•Ҡмңјл©ҙ м•ҢлӢӨмӢңн”ј Object.equals()лҘј мӮ¬мҡ©н•ҳкё° л•Ңл¬ём—җ лҸҷмқјм„ұмқ„ л№„көҗн•ҳкІҢ лҗңлӢӨ

  • equals()м—җм„ң trueк°Җ л°ҳнҷҳлҗңлӢӨл©ҙ л‘җ к°қмІҙмқҳ hashCode()лҠ” л°ҳл“ңмӢң к°ҷм•„м•ј н•ңлӢӨ
  • equals()м—җм„ң falseк°Җ л°ҳнҷҳлҗҳлҠ” л‘җ к°қмІҙмқҳ н•ҙмӢңмҪ”л“ңк°Җ к°ҷм•„лҸ„ мғҒкҙҖм—Ҷм§Җл§Ң, лҗҳлҸ„лЎқ лӢӨлҘё н•ҙмӢңмҪ”л“ңлҘј к°Җм ём•ј н•ңлӢӨ
    • м„ңлЎң лӢӨлҘё к°қмІҙк°Җ к°ҷмқҖ н•ҙмӢңмҪ”л“ңлҘј к°–кІҢ лҗҳл©ҙ н•ҙмӢң мһҗлЈҢкө¬мЎ°м—җм„ң мӨ‘ліөмқҙ л§Һм•„м ё кІҖмғүмҶҚлҸ„к°Җ л–Ём–ҙм§„лӢӨ
